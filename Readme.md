# Simple Textarea Character Counter

A lightweight web app that demonstrates a **character counter** for a textarea input with a maximum limit of 200 characters. It provides real-time feedback to the user, showing both the number of characters typed and the number remaining. When the limit is reached, a warning message is displayed.

---

## ✨ Features

- **Live character count**: Updates as the user types.
- **Remaining characters**: Shows how many characters are left out of 200.
- **Warning message**: Displays a red warning when the user reaches the limit.
- **Responsive design**: Styled with CSS for a clean, centered card layout.
- **Accessible feedback**: Clear text indicators for usability.
